# Prior work versus Build Week work

This disclosure compares `main` with the `OpenAI-build-week-2026` submission
branch. It is intended to let judges distinguish the existing foundation from
the work produced for OpenAI Build Week 2026.

## Prior foundation

Before the competition branch, the repository already contained a general
game-analysis agent, Godot command-line integration, simulation/report
pipelines, a basic report browser, and adapters for local OpenAI-compatible
model servers. The Study in Germany game concept also existed outside the
submission workflow.

## Built or substantially rebuilt for Build Week 2026

- the provider-neutral Persona gateway with hash-pinned Replay and an explicit
  server-side OpenAI Responses API path;
- the six-Persona, three-seed campaign contract and the committed real-Godot
  evidence matrix;
- causal repair experiments with fixed and unseen-holdout cohorts, including
  the retained rejected candidate;
- the repository-scoped `playtest-forge` Codex Skill and offline Judge CLI;
- the embedded, hash-inventoried demo snapshot and portable judge image;
- Judge API safety boundaries, fail-closed G0–G5 review gates, and platform
  evidence contracts;
- the Judge Mission and Playthrough Inspector competition experience,
  strategy-persona artwork, two-frame replay movement, and the unified
  auxiliary-page visual system;
- the submission, review, video, attribution, and release-readiness material.

## Codex and GPT roles

Codex was the central development and repair director: it inspected evidence,
formed the bounded hypothesis, implemented the isolated candidate, ran fixed
and holdout verification, and rejected the candidate when the evidence did not
support it. The retained core task ID is
`019f6816-fe0d-78a2-9001-1890b32ef820`.

GPT-5.6 is the optional live Persona decision provider through the OpenAI
Responses API. The committed playthrough evidence remains truthfully labeled
deterministic Replay until a separate restricted-key live run is completed and
its sanitized evidence is imported. No live claim is inferred from Replay.
